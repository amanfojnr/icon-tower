


# icon-tower
An open source svg icon pack  for your projects.

## How-To 
You can use any svg editor to grab the icons you need and also cutomize the colors to 
fit your use case.

#### Recommended sotware to use
- Inkscape(opensource)
- Adobe Illustrator
